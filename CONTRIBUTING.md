# Contributing

## Development Setup

```bash
npm install
npm run lint
npm test
npm run start
```

## Pull Request Requirements

1. Keep changes small and focused.
2. Add tests for backend behavior changes.
3. Ensure lint and tests pass before requesting review.
4. Update `CHANGELOG.md` for notable user-facing changes.

## Security Expectations

1. Do not commit secrets or production credentials.
2. Validate all API input fields and transitions.
3. Keep dependency changes minimal and reviewed.
